Project Statement- Stone, Paper, Scissors Game

Problem Statement To create a simple, interactive, command-line game of Stone, Paper, Scissors that a single user play against the computer. The game must determine the winner based on the rules and provide feedback on the outcome and the choice.

Scope of the Project The scope of this project is strictly limited to a single player command line interface application.

In Scope

Implementing the game logic for win, loss, or draw.
Generating a choice for the computer player.
Handling input and validating it against the choices.
Providing clear output for the result of each round.
Out of Scope 1.Graphical User Interface 2.Tracking and displaying the user score in multiple rounds. 3. Adding many players function. 4. Inputting different game modes or difficulty.

Target Users The primary target users for this game are

Beginner Programmer learning Python or basic game development who can use the code as a simple example. Casual Users Anyone looking for a quick and simple game to play on their computer through the command line.

High Level Features

1.Random Computer Choice The system will use the random module and select one of the three options 2.Interactive User Input The user can input its choice by the command line. 3. Input Validation The game will check if the user's input is one of the correct choices and tell the user to try again if it is wrong. 4.Game Logic Engine The main function to compare the user's choice with the computer's choice and tell the right result. 5. Clear Result Feedback The game provides output telling the result of the round along with the computer's answer.
